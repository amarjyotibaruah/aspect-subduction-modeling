# Figures

This folder contains selected visualization outputs from ASPECT geodynamic models used to illustrate mantle flow patterns and thermal structure.

## Contents

### Simple Subduction Model
- `3D_simple_subduction_isoviscous_1e20PaS_velocity_field.jpeg`
  - Shows large-scale poloidal flow and edge-driven flow in a simple subduction system.

### Small-Scale Convection (SSC) Model
- `3D_simple_subduction_isoviscous_1e20PaS_drip_zone_1e18PaS_4Myr_velocity_vectors.jpeg`
  - Displays development of small-scale convection beneath the lithosphere.
- `3D_simple_subduction_isoviscous_1e20PaS_drip_zone_1e18PaS_4Myr_temperature.jpeg`
  - Highlights thermal structure and regions of mantle instability.

### Slab-Edge Toroidal Flow Model
- `3D_slab_edge_toroidal_flow_isoviscous_1e20PaS_velocity_vectors.jpeg`
  - Shows toroidal mantle flow around slab edges in a 3D geometry.
- `3D_slab_edge_toroidal_flow_isoviscous_1e20PaS_temperature_field.jpeg`
  - Illustrates slab thermal structure and lateral temperature gradients near slab edges.

## Notes

These figures are representative outputs selected to demonstrate key physical processes including poloidal flow, small-scale convection (SSC), and toroidal circulation. Additional model outputs are part of ongoing research work.
